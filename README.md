# Analyzing the Impact of AI-Assisted Diagnostics in Healthcare Operations

## Business Question

How does implementing AI-assisted diagnostic technology improve healthcare efficiency, accuracy, and clinical workload?

## Dataset

Synthetic healthcare dataset simulating AI diagnostic implementation.

## Tools

Excel
Pivot Tables
Data Cleaning
Dashboard Design

## Findings

AI implementation reduced average turnaround time from 19 hours to 9.6 hours, improving efficiency by approximately 49%
